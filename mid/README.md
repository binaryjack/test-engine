# MID-Level React Developer — Exam Overview

## Exam Format
- **40 multiple-choice questions** — scenario-based, not just trivia
- **105 minutes of coding challenges** — real editing in a browser IDE
- Topics focus on intermediate React: hooks mastery, patterns, performance, error handling

## What "Mid-Level" Means on This Exam
You are expected to know React well enough to:
- Write production-quality components with correct hook usage
- Choose the right pattern for the problem (not just the first one that works)
- Understand WHY React behaves the way it does (rendering model, reconciliation)
- Handle real-world concerns: loading states, errors, forms, performance

## Key Mindset for the Exam
- Always ask: **"Who owns this state?"**
- Always ask: **"What triggers a re-render here?"**
- Always ask: **"What happens on cleanup?"** (useEffect)
- Always ask: **"Is this the simplest solution?"**
