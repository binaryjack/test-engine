Exam Levels
To provide the most accurate information on your skill level, we’ve broken up the certification into 3 different levels, each with its own exam.

Level 1 - Certified Junior-Level React Developer
Passing level one means you know the very basics of React. Effectively, you'd pass as a great hire for a junior React dev position.

Level 2 - Certified Mid-Level React Developer
Passing level 2 means you're familiar with more advanced concepts and strategies and have a better grasp of practical React patterns and strategies. Those who pass level 2 would be general hires.

Level 3 - Certified Senior-Level React Developer
Passing level 3 means you are at the top of the React developer pool. You are familiar with the most advanced concepts and can architect large-scale apps with React. Those who pass level 3 would be great hires for senior-level positions.
