# SENIOR-Level React Developer — Exam Overview

## Exam Format
- **40 multiple-choice questions** — architecture-level, React internals, React 19
- **105 minutes of coding challenges** — more complex real-world scenarios
- Builds on MID-level knowledge — master MID topics first

## What "Senior" Means on This Exam
- Deep understanding of React's rendering model and concurrent mode
- Able to architect large-scale applications with React Server Components
- Proficient with React 19's new APIs (Actions, useActionState, useOptimistic)
- Performance expert: knows when and why things re-render
- Can make technology decisions (when to use what state mechanism, when RSC vs CSR)

1. [React 19 Actions](./01-react-19-actions/README.md) — useActionState, useOptimistic, form actions, use()

## Mindset Shift from MID to SENIOR
- MID: "Does this code work?"
- SENIOR: "Is this the right approach? What are the tradeoffs? What happens at scale?"
